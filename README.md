# [WIP] Chewy::Resque

Resque atomic & urgent strategies for [Chewy](https://github.com/toptal/chewy) gem.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'chewy-resque'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install chewy-resque
